Instructions

  1. Install your biometric software (e.g. ZKTeco Attendance Management) on this computer.
  2. Click Browse and select the database file, usually named att2000.mdb, inside the install folder — or paste the full path manually.
  3. Slashes are normalized automatically to match what the backend expects, so it doesn't matter if you type / or \.
  4. Click Save & Link to send the path to the attendance server and connect the device.
  5. Make sure the machine is powered on and connected to the same network.
Note: Because of browser security restrictions, the file picker on its own cannot reveal the full folder path — only the file name. After browsing, double-check (or complete) the folder portion of the path before saving.
